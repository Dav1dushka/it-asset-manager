# IT Asset Manager

A simple web application for managing IT assets and devices.


##Screenshot
<img width="1117" height="601" alt="image" src="https://github.com/user-attachments/assets/be783b80-c765-4ccf-9da8-d2995b55be6f" />

## Features

- Add devices
- Delete devices
- Search devices
- Store data in Local Storage
- Track IP addresses
- Track device status

## Technologies

- HTML5
- CSS3
- JavaScript

## Purpose

This project simulates a small IT inventory management system used by IT technicians and system administrators.
