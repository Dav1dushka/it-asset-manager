# IT Asset Manager

A simple web application for managing IT assets and devices.


##Screenshot
<img width="1283" height="630" alt="image" src="https://github.com/user-attachments/assets/13a489cb-a63a-43f8-9077-fcb7253d601d" />


## Features

- Add devices
- Delete devices
- Search devices
- Store data in Local Storage
- Track IP addresses
- Track device status

## Technologies

- HTML5
- CSS3
- JavaScript

## Purpose

This project simulates a small IT inventory management system used by IT technicians and system administrators.
